# BG3 Character Builder

A Python-based Baldur’s Gate 3 character builder for planning classes, subclasses, and level progression.

## Features

- Choose a class and subclass
- Model level progression from 1–12
- View a summary of your build
- Designed as a personal learning and portfolio project

## Running the Project

Make sure you have Python installed, then run:

python main.py


## Future Plans
